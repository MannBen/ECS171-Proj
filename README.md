# ECS171-Proj
To run Pokemon interface code,
1. pip install flask (Prevents --> ModuleNotFoundError: No module named 'flask')
2. pip install beautifulsoup4 (Prevents --> ModuleNotFoundError: No module named 'bs4')
3. Install other packages if error stating ModuleNotFoundError
4. cd Interface 
5. Run "python main.py"
6. When see this statement, "Running on http://127.0.0.1:5000", copy the link http://127.0.0.1:5000 and paste URL in address bar.
7. Once entered, interface should work.
